TypeScript Fastify API for workout plans with Better Auth and Prisma/PostgreSQL. Entry point: src/index.ts. Routes live in src/routes, use cases in src/usecases, schemas in src/schemas, shared infra in src/lib, errors in src/errors. Prisma schema is in prisma/schema.prisma and generated client code is written to src/generated/prisma and should not be edited manually. Local database is defined in docker-compose.yml.
